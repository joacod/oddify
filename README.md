## 👋 Introduction

An odd looking UI component library for [Astro](https://astro.build/) 🚀

## 💻 Development

**Install library:**

```bash
npm i oddify
```

**Configure tsconfig.json in your project:**

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@oddify": ["../lib.astro"]
    }
  }
}
```

**Use component example:**

```bash
---
import {Button} from "@oddify";
---

<Button>Click Me</Button>
```
